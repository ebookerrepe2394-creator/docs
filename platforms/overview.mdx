Addictiva Records funciona como el centro de mando para tu carrera musical: en lugar de iniciar sesión en cada plataforma por separado, conectas todas tus cuentas una sola vez y gestionas distribución, estadísticas y lanzamientos desde un único dashboard. Ya sea que quieras subir un nuevo sencillo a DistroKid, revisar tus streams en Spotify o monitorear tus oyentes en Apple Music, todo ocurre en el mismo lugar, sin cambiar de pestaña ni recordar decenas de contraseñas.

## Tipos de conexión por plataforma

Cada plataforma utiliza un método de autenticación distinto. La siguiente tabla resume cómo Addictiva Records se conecta con cada servicio y qué nivel de acceso se establece:

| Plataforma      | Tipo de conexión | Acceso de lectura | Acceso de escritura |
| --------------- | ---------------- | :---------------: | :-----------------: |
| Spotify         | OAuth 2.0        | ✓                 | —                   |
| Apple Music     | API Key (MusicKit) | ✓               | —                   |
| YouTube Music   | Google OAuth 2.0 | ✓                 | —                   |
| SoundCloud      | OAuth 2.0        | ✓                 | ✓                   |
| DistroKid       | API Key          | ✓                 | ✓                   |
| TuneCore        | API Key          | ✓                 | ✓                   |

**OAuth 2.0** redirige al artista a la página oficial de la plataforma para autorizar el acceso sin compartir credenciales con Addictiva Records. **API Key** requiere generar una clave desde la configuración de la plataforma y pegarla en tu dashboard de Addictiva Records.

<Note>
  Conectar una plataforma a Addictiva Records otorga acceso de lectura para importar estadísticas y, en las plataformas que lo permiten, acceso de escritura para publicar contenido y gestionar lanzamientos. Puedes revocar el acceso en cualquier momento desde **Configuración → Plataformas** sin afectar tu cuenta en la plataforma original.
</Note>
